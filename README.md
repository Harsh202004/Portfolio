# Harsha Rajesh Hutagi - Portfolio

A modern, recruiter-focused personal portfolio website built with HTML, CSS, and JavaScript.

## Live Project
- GitHub Repo: [Harsh202004/Portfolio](https://github.com/Harsh202004/Portfolio)

## Highlights
- Clean single-page portfolio layout
- Sections for About, Projects, Skills, Timeline, Gallery, and Contact
- Certificate gallery with fullscreen lightbox
- Animated 3D visuals using Three.js
- Scroll and interaction animations using GSAP

## Project Structure
```text
.
|- index.html               # Main page structure
|- styles.css               # All styling
|- app.js                   # All interactive behavior
|- portfolio-data.json      # Strict JSON data (no comments)
|- portfolio-data.jsonc     # Editable data with comments
|- Harsha_Hutagi_Resume.pdf # Resume file linked in UI
|- images/
   |- gallery/
      |- certificate-illinois-tech.png
      |- certificate-cdac-security.png
      |- certificate-aws-simplilearn.png
```

## Local Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh202004/Portfolio.git
   ```
2. Open the project folder.
3. Launch `index.html` in a browser.

For best results, run via a local server (VS Code Live Server, etc.).

## Update Content
- Edit `portfolio-data.jsonc` for easier maintenance with visible comments.
- Keep `portfolio-data.json` for strict JSON usage when needed.
- Replace certificate images in `images/gallery/` and keep paths updated.

## Deployment
You can deploy this as a static site on:
- GitHub Pages
- Netlify
- Vercel
